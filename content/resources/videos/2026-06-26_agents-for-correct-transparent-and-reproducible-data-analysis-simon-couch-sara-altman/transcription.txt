<h3>Why correctness matters in data analysis</h3>

<p><a data-t="1380">Okay. So, why is this a problem? I think probably all intuitively understand that correctness matters in data analysis. <p><a data-t="1962">So we also want to design the scaffold or the harness so that it matters less when they are wrong.</a> <a data-t="1969">I'm going to talk about two things.</a> <a data-t="1973">The first is sort of using code as a foundation. <h3>Pausing to involve the user</h3>

<p><a data-t="2256">So we talked about having the agent write code, showing it to the user, and working in the same environment. <p><a data-t="2313">And this is in contrast to how it could behave, which is just, you know, you ask it a question and it goes off for as long as it thinks is necessary to completely solve the problem.

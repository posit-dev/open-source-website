<p><a data-t="0">If you create a ggplot without specifying any unique aesthetics or a theme, you'll notice the</a> <a data-t="5">package still makes some choices for you.</a> <a data-t="10">Ever wonder what those choices are? Because each geom</a> <a data-t="10">has a different set of default aesthetics, and you might be surprised to find out that they're</a> <a data-t="15">almost all based on the source code for a specific theme that you haven't even applied.</a></p>

PyCascades is a regional Python software conference in the Pacific Northwest, celebrating the west coast Python developer and user community. The 2026 conference takes place March 20-23 in Vancouver, British Columbia.

Rodrigo Silva Ferreira from Posit will be presenting at the conference, sharing insights about Positron, the new IDE for data science. PyCascades brings together Python developers, data scientists, and enthusiasts from across the Pacific Northwest region to learn, share, and connect.

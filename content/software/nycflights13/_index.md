The nycflights13 package provides a comprehensive dataset of all 336,776 flights that departed from New York City airports (JFK, LaGuardia, and Newark) in 2013. Designed as a teaching tool and research resource, this package offers real-world aviation data that's perfect for learning data manipulation, visualization, and statistical analysis in R. Whether you're exploring patterns in flight delays, analyzing seasonal trends, or building predictive models, nycflights13 provides a rich, accessible dataset that demonstrates practical data science techniques.

Beyond raw flight records, the package includes five interconnected datasets that enable deep analysis of flight performance. The flights table contains departure and arrival details for every flight, while supplementary tables provide hourly weather data for each airport, aircraft construction information, airport locations and names, and airline code translations. This integrated structure makes it easy to investigate questions like how weather conditions affect delays, which aircraft models perform best, or how different airlines compare across routes. The package's clean, well-documented data has made it a staple in data science education and a go-to resource for demonstrating data wrangling and exploratory analysis techniques.
